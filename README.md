## 협업
| 타입  | 설명                                   |
|------------|----------------------------------------|
| `feat`     | 새로운 기능 추가                       |
| `fix`      | 버그 수정                              |
| `docs`     | 문서 수정                              |
| `style`    | 코드 포맷팅, 세미콜론 누락 등 코드 변경이 없는 경우 |
